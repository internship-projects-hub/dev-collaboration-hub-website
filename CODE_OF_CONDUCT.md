# Code of Conduct

## Our Standards

We are committed to providing a welcoming and respectful environment for all contributors.

Examples of positive behavior include:

- Being respectful and professional
- Giving constructive feedback
- Helping other contributors learn and grow
- Collaborating in good faith

Examples of unacceptable behavior include:

- Harassment or discrimination
- Personal attacks
- Offensive language
- Disruptive or unprofessional conduct

## Enforcement

Project maintainers have the right to remove comments, issues, pull requests, or contributors who violate this Code of Conduct.

## Scope

This Code of Conduct applies to all project spaces, including GitHub repositories, discussions, issues, pull requests, and community communications.

## Contact

For questions or concerns, contact the project maintainers through GitHub Discussions or Issues.
